# sistema-web-odontologico-univesp

🦷 Sistema de Estoque Odontológico – UNIVESP

Projeto desenvolvido para controle de estoque odontológico, permitindo o cadastro e gerenciamento de produtos utilizados em clínicas odontológicas.

Este sistema foi desenvolvido como parte das atividades acadêmicas da UNIVESP, utilizando Django no backend e HTML/CSS no frontend.

📌 Funcionalidades
Cadastro de produtos odontológicos
Listagem de produtos em estoque
Edição de produtos
Remoção de produtos
Controle básico de quantidade em estoque
Interface web simples e funcional

🛠️ Tecnologias Utilizadas
Python
Django
HTML
CSS
SQLite (banco de dados padrão do Django)

📌 Objetivo do Projeto

Este sistema foi desenvolvido com o objetivo de:

Aplicar conceitos de desenvolvimento web com Django
Criar um sistema funcional de controle de estoque
Praticar organização de projetos web
Integrar backend e frontend
